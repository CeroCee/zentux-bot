Place your PayPal QR image in this folder as:

  paypal-qr.png

The bot will automatically use this file when the user presses the "Dinero" button on /compra.

Alternatively, you can configure either of these in your .env:
- PAYPAL_QR_PATH=C:\\absolute\\path\\to\\your\\qr.png
- PAYPAL_QR_URL=https://your.domain/qr-paypal.png
